# Privacy Policy — WHOOP Personal Data Pipeline

**Last updated:** [07-8-26]

This is a personal, non-commercial software project built by [YOUR NAME] as
part of a data engineering/data science portfolio. It is not a commercial
product and is not distributed to other users.

## What data this app accesses

Using the WHOOP API, this project accesses the following data belonging
solely to the developer's own WHOOP account:

- Recovery scores, heart rate variability, and resting heart rate
- Physiological cycle (strain) data
- Sleep data (duration, stages, performance)
- Workout data (activity type, strain, heart rate)
- Basic profile and body measurement information

## How the data is used

Data is pulled via the WHOOP API and stored locally / in a private
GitHub repository owned by the developer, for the sole purpose of
personal analysis, visualization, and demonstration in a portfolio
project (e.g., dashboards, SQL practice, exploratory analysis).

## Data sharing

This data is **not** sold, shared, or made available to any third party.
It is not used for advertising. Any dashboards or write-ups produced from
this data will only include the developer's own, personal metrics unless
explicitly stated otherwise.

## Data retention

Data is retained locally / in the developer's private repository for as
long as the portfolio project is maintained. The developer can revoke this
app's access at any time via WHOOP's account settings, which stops all
further data collection.
